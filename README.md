# Dodge the Creeps

A simple 2D survival game made with the Godot Engine where the player avoids enemies for as long as possible to get the highest score.

## About the Game

In this game, the player controls a character that must dodge incoming enemies ("creeps").  
The longer you survive, the higher your score becomes.

This project was created using Godot and is based on the classic beginner tutorial project.

---

## Features

- Player movement using keyboard controls
- Enemy spawning system
- Score tracking
- Game over screen
- Restart functionality
- Sound effects and animations

---
